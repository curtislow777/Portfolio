Frog Character

Project Goals

This projects shows the workflow used for creating my 3D character, from initial concept and design on a model sheet to a fully animated model within Unity.

Conceptualise and devlopping a model sheet, transform the 2D model sheet into a 3D character through modelling and sculpting, retopology for auto rigging purposes, adjusting skin weights, texturing in substance painter, and integration into unity with mixamo animations

Tools Used

Maya
Substance Painter
Unity


Overview
Background

This project started as a 3D character creation assignment for school. We had the freedom to design any bipedal character we wanted, so I decided to create a humanoid frog for a fun and unique twist. Instead of the typical human design, I wanted to explore something that sparked my interest, like a character you might see in a cartoon.

Sci-Fi Environment

Project Goals

To create and utilize modular assets, trim sheet texturing with high-to-low poly baking and the integration of VFX and post processing effects.

Tools Used

Maya
Substance Painter
Unity


Overview
Background

This project started was done for an assignment in school. This was one of my first projects using Unity and assembling a real-time environment. With a sci-fi theme, the possibilities for creative VFX were wide open. It was also my first attempt at creating a trim sheet and tilable textures. Mastering this workflow has deepened my understanding of the process behind creating games and environments.

Stonk Nook

Project Goals

To integrate firebase database and authentication in a Unity game.

Technology Used

Unity
C#
Firebase Authentication
Firebase Database


Overview
Background

This project was developed as part of a school assignment focused on integrating a real-time database with a Unity-based game. The objective was to create a backend system capable of handling robust database interactions and user authentication, providing the freedom to choose any game concept for this integration. I was inspired by the game pac-man and decided to recreate it but with Tom Nook from animal crossing as the main character. The pellets in the game are replaced by bitcoins for Tom Nook to harvest and build his money empire.
