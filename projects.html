Frog Character

Project Goals

This projects shows the process of creating my 3D character, beginning with an initial concept and design on a model sheet.The character was transformed from a 2D design into a 3D model through various stages including modeling, sculpting, and retopology for rigging. Further adjustments were made to skin weights and texturing using Substance Painter, culminating in the integration of the character into Unity with animations from Mixamo.

Tools Used

Maya
Substance Painter
ZBrush
Unity
Mixamo


Overview
Background

This project started as a 3D character creation assignment for school. We had the freedom to design any bipedal character we wanted, so I decided to create a humanoid frog for a fun and unique twist. Instead of the typical human design, I wanted to explore something that sparked my interest, like a character you might see in a cartoon.

Mother Nature Dashboard

Project Goals

The goal of this project is to design and implement a scalable Firebase database integrated with our VR game Mother-Nature and a real-time interactive web dashboard. This involves Firebase Authentication services, Firebase Realtime Database for live data updates within Unity, and Firebase Storage for multimedia content management. The web dashboard, developed using JavaScript, will display these updates dynamically and player statistics in real-time.

Tools Used

HTML
CSS
Javascript
Bootstrap 5
Firebase Database
Firebase Authentication
ChartJS


Overview
Background

This webpage is an add-on to complement our VR game Mother-Nature. The dashboard will display information that is useful to the admin of the game. It features visual representations such as line charts for tracking user growth, a leaderboard, activity logs for player achievements, and a pie chart detailing player demographics. Additionally, it includes metrics on game completion and checkpoint progression, alongside a table for detailed player information with the capability to view in-game images and add new players through a signup form. This dashboard enhances the administrative capabilities by offering real-time insights and streamlined management functionalities to improve player engagement and game operations.

Project Goals

This project aims to create a 30-second motion graphics clip showcasing popular landmarks in a chosen country. It involves storyboarding, animation, and motion effects to highlight key attractions.
